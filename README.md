# zython

A fast python interpreter written in Zig.


## Features

**Data types**
- numbers
- booleans
- strings


**Operators**
- Arithmetic operators (+, /, **, etc)
- Logical operators (not, and, or)
- Bitwise operators (except shifts)
- Comparison operators (<, ==, etc)
- Walrus operator


**Built-ins**
- print


**Misc**
- If-else statements
- Pass
- While loops, break, continue
- Global variables

See `/examples`.
